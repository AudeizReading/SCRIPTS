 94 " auto-format fichier .c : utilise le cindent, formatoptions permet d'utiliser
 95 " le textwidth (sinon j'ai des soucis avec en C, ne se coupe pas à la col 80) et
 96 " formatte les commentaires C de la façon suivante 
 97 " /* 
 98 " ** Ceci est un commentaire C
 99 " */
100 " Pour la dernière ligne de commentaire, comme il ne sait pas encore que c'est
101 " la dernière ligne, elle commencera par **, le simple fait de taper / annule **
102 " le commentaire se terminera alors
103 "autocmd FileType c set cindent formatoptions=tcroq comments=sl:/*,mb:**,elx:*
104 autocmd FileType c set cindent formatoptions=tcroq comments=sl:/*,mb:**,elx:*/

108 " (www.davide.blanchet.free.fr/vim/doc/fr62017/autocmd.txt.html, *skeleton*)
109 " :autocmd BufWritePre, FileWritePre *.html     ks|call DerModif()|'s
110 " :fun DerModif()
111 " :     if line("$") > 20
112 " :         let l = 20
113 " :     else

116 " :     exe "1," . l . "g/Dernière modification : /s/Dernière modification :
