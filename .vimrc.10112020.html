 90 " auto-format fichier .c : utilise le cindent, formatoptions permet d'utiliser
 91 " le textwidth (sinon j'ai des soucis avec en C, ne se coupe pas à la col 80) et
 92 " formatte les commentaires C de la façon suivante 
 93 " /* 
 94 " ** Ceci est un commentaire C
 95 " */
 96 " Pour la dernière ligne de commentaire, comme il ne sait pas encore que c'est
 97 " la dernière ligne, elle commencera par **, le simple fait de taper / annule **
 98 " le commentaire se terminera alors
 99 "autocmd FileType c set cindent formatoptions=tcroq comments=sl:/*,mb:**,elx:*
100 autocmd FileType c set cindent formatoptions=tcroq comments=sl:/*,mb:**,elx:*/

102 " Il est aussi possible de modifier la date et l'heure de modification dans un
103 " fichier donné , exemple avec un fichier .html :
104 " (www.davide.blanchet.free.fr/vim/doc/fr62017/autocmd.txt.html, *skeleton*)
105 " :autocmd BufWritePre, FileWritePre *.html     ks|call DerModif()|'s
106 " :fun DerModif()
107 " :     if line("$") > 20
108 " :         let l = 20
109 " :     else
110 " :         let l = line("$")
111 " :     endif
112 " :     exe "1," . l . "g/Dernière modification : /s/Dernière modification :
113 " .*/Dernière modification : " . \ strftime("%d %b %Y %X %Z")
114 " :endfun
